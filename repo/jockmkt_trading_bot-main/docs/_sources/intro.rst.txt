============
Introduction
============

Motivations
===========

- The JockBot was developed with the intention of making it easier for technical users to implement electronic trading strategies.

Strengths
=========

- Organizes websocket messages asynchronously
- Stores event data as far as the user wants to look back
- Includes multiple trading strategies, for both live and ipo phases.
- Includes a framework for a user to easily build their own strategy classes

Limitations
===========

- The end user must have technical expertise to implement this code.
