StrategyConfig Class

class strategy_base.StrategyConfig

Base strategy config class

Variables

  • event_id – the event ID for the event the user wishes to trade in

  • order_size – number of shares per order. some strategies allow you to set the $ amount too. see that strategy’s config. Default: 1

  • api_key – your api token

  • secret_key – your secret key

  • max_spend – the max $ amount you want to spend. Default: 1000

  • max_position – the maximum number of shares you would like to own at a time

  • phase – a list of phases, must contain at least one of ‘ipo’ or ‘live’ depending on your strategy, or both.

  • array_len – how much historical data the user would like saved. An array of length 1000 will hold the last 1000 updates, for example, to projected fantasy points (or other statistics/metrics) default: 300

  • print_to_console – should logging messages be printed to console

  • log_path – where the user wishes logs to be stored. The default will create a log folder wherever the bot is being run from. default: ‘./’

  • log_level – logging.DEBUG, logging.INFO, logging.ERROR, etc. Default: logging.DEBUG or 10

  • log_path – the path you’d like to build your logs folder. Default ‘./’

  • always_update – If false, numpy arrays for each player will only update when there is a change in that data point (i.e. only when a player scores fantasy points, or their estimated price changes). If true, numpy arrays update regardless of whether there is a change in fpts, price, or stat. Default: False

  • prefill – if True, numpy arrays will be filled with the initial statistic, price or metric. If False, fills numpy arrays with zeros, and the last index of the array will have initial stat, price or metric. Default: True

  • message_consumers – number of websocket message consumers. The more players there are in the event, the more you should increase this number. Max recommended is 12.

  • port – the port for web interface, default: 8080. Can be set via env vars

  • host_name – host name, defaults to 127.0.0.1 (localhost)

  • web_popup – open up a webpage with information about holdings and pnl. Default: True
