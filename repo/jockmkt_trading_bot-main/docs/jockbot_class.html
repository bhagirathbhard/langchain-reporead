JockBot Structure

information

  • The JockBot is built on top of the jockmkt-sdk

class bot.JockBot(client: jockmkt_sdk.client.Client, strategy: jockbot.strategy_base.Strategy, counter: bot.EnvOrderCt)

Class responsible for most of the functionality of the jockbot. Not all attributes will be annotated.

Parameters

  • client – Instance of jockmkt_sdk.client.Client

  • strategy – Instance of strategy_base.Strategy

Variables

  • fees – current fee amount to trade. 0.02 or 2% at this release

  • client – stored version of client arg for access within a strategy

  • event_live – bool, whether the event is open to trading

  • event_info – jockmkt_sdk.objects.Event containing info about the event in which we are trading

  • event_status – str, the event’s current status

  • player_dict – The entire player dictionary for the event, keyed by tradeable_id, structured as follows:

player_dict = {
    'tdbl_xxx': {
        'last_updated': timestamp,
        'tradeable': Tradeable object (see Jockmkt SDK docs for more)
        'count_ticks': int, the number of times this tradeable has been updated,
        'timestamp': np.ndarray[timestamp],
        'pregame_projection': pregame fpts projected, float
        'fpts_projected': np.ndarray[float] array of live fpts projected
        'fpts_scored': np.ndarray[float] of fpts scored
        'estimated': np.ndarray[float] of estimated prices
        'bid': np.ndarray[float] of bids
        'ask': np.ndarray[float] of asks
        'last': np.ndarray[float] of last traded price
        'high': np.ndarray[float] of high traded price
        'low': np.ndarray[low] of low traded price
        'stats': {
            'stat1': np.ndarray[float or int] of a stat
            the stats dict differs between sport, but contains all numerical stats for each player
        }
    }
}
Variables

  • public_orders – a list of orders placed by the public

  • public_trades – a list of trades placed by the public

  • games – dict keyed by game id, mapping to jockmkt_sdk.objects.Game objects for all relevant games in the event

  • teams – dict keyed by team_id, mapping to jockmkt_sdk.objects.Team objects for all teams in event

  • balance – float, available balance.

  • holdings – dict mapping tradeable_id to jockmkt_sdk.objects.Tradeable for players in event

  • entry – jockmkt_sdk.objects.Entry object

  • my_orders – list of currently active orders
